Classification Module

The information is organized in such a way that for each hierarchical level (Order, Suborder and Great Group) it must start from the first element of the list and in case of not corresponding to the soil to be classified, continue with the next element.

Example: the Orders list follows the sequence:
Gelisols -> Histosols -> Spodosols -> Andisols -> ...

It starts with the Gelisols and it checks if the soil to be classified meets the requirements of Gelisol, in case of not fulfilling them, it continues with the Histosols and so on until the corresponding Order is found. Once the Order is defined, the same procedure is followed to define the Suborder and then the Great Group.

In the initial screen of each Soil Order, in addition to the definition, there is a series of slides that includes: a photo of a soil profile, a global distribution map of that soil Order and a map of that Order in Argentina.
